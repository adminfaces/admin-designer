Showcase menu This page desmonstrates showcase menu and top bar code

AdminFaces Showcase Application

The menu is defined via facelets templating by *extending* Admin Template.

Note

Menu items are activated (class active) based on current url and menu href.

Note

To use horizontal menu layout use admin-top.xhtml template.

Note

To use central title on top bar (as in this page) just define center-header on you page or template.

Tip!

You can customize central title by using a css overide on class center-navbar:


.body .center-navbar {
  color: red;
}

Important!

Center title is hidden on mobile (small screens).
